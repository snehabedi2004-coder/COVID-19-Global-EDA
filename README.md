\# COVID-19 Global Data Analysis (EDA)



\## Project Overview

Performed exploratory data analysis on WHO COVID-19 global dataset to uncover trends in COVID-19 cases and deaths across countries and regions.



\## Tools \& Technologies

\- Python

\- Pandas

\- Matplotlib

\- Jupyter Notebook



\## Key Analysis Performed

\- Calculated Death Rate for each country and identified top 10 severely affected countries

\- Analyzed monthly trends globally and for India to identify outbreak peaks

\- Conducted region-wise comparison using WHO regions

\- Visualized insights using bar and line charts for clear interpretation



\## Dataset

Source: \[World Health Organization (WHO)](https://www.who.int/data)



\## Outcome

This project highlights the countries and regions most affected by COVID-19 and provides insights into global pandemic patterns.
